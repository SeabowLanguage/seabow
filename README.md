# Seabow Language

## What is Seabow Language ?

> Seabow is a programming language written in C++ which provides much more readable syntax. It also includes an interpreter, a bytecode generator and a bytecode runner. It is created to provide a brand new language who can be written in one Operating System and run in multiple other Operating Systems. Seabow can manage compile-time and runtime errors, macros, non-nullable values and much more.

## How can I use Seabow ?

> You can use Seabow simply by download a seabow release and read the seabow documentation [here](https://seabowlanguage.github.io/).
